class lib.core.xtream.XtreamClient(host, port)¶

XtreamCodes IPTV Player HTTP API v2 wrapper

Inheritance:

lib.common.client.BaseClient()
Params:
host:(str) server domain e.g “http://www.your-iptv.net”
port:(int) server port number (0-65535)

Note

XtreamClient does not store a session username / password, consequently these values must be passed to any endpoint requests that require authentication.

XtreamClient makes the following methods available for:

  • Validating supplied user credentials.
    • require (auth) get_account_info()
  • Executing content queries to the server.
    • require (auth) get_category_list()
    • require (auth) get_section_index()
    • require (auth) get_category_index()
    • require (auth) get_series_content()
    • require (auth) get_series_index()
    • require (auth) get_archive_index()
    • require (auth) get_stream_info()
  • Computing playable stream URLs.
    • archive_url()
    • stream_url()

Note

If an account is cancelled or expired we want the user to know immediadely, not when the cache expires, so get_account_info() sets the _cache=False flag on get_payload()
