common.console¶

lib.common.console¶

lib.common.console.printError(message)¶

Prints message using colored format with leading and trailing newline to STDERR

Params:
message:(str) color encoded string

Note

uses string() internally.

lib.common.console.printLine(message)¶

Prints message using colored format with trailing newline to STDOUT

Params:
message:(str) color encoded string

Note

uses string() internally.

lib.common.console.string(text)¶
Returns colored string using python replacement. Available Colors:
{W} 033[0m White
{R} 033[31m Red
{G} 033[32m Green
{O} 033[33m Orange
{B} 033[34m Blue
{P} 033[35m Purple
{C} 033[36m Cyan
{GR} 033[37m Gray
{D} 033[2m Dims current color
Params:
text:(str) color encoded string
Returns:

str
Example¶
console.string("{R}This text is red.{W} This text is white")
>>> u'\033[31mThis text is red.\033[0m This text is white'
