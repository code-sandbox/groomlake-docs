common.client¶

lib.common.client¶

class lib.common.client.BaseClient(host, port)¶

Base class containing common logic for setting up a session with a web server, as well as caching / providing cached responses if possible. Also ensures some basic URL conformity.

Params:
host:(str) server domain
port:(int) server port number (0-65535)
Throws:

ConfigurationError()

Note

subclasses are responsible for populating API data.

static get_payload(path, _cache=True)¶

defers data retrieval latency / bandwidth to local storage using lib.common.cache. Avoiding the slower lib.common.client until it’s necessary to retrieve new data or refresh a stale payload.

Params:
path:(str) URL for target desired remote resource
_cache:(bool) enable / disable lib.common.cache
Returns:

str(payload) or None

lib.common.client.randomagent()¶

Sometimes you need to get stuff from servers that will only talk to browsers this will spoof a random browser string to use in place of the urllib User-Agent.

Returns:pseudo random User-Agent string
lib.common.client.request(url, timeout=30, randua=False)¶

http(s):// GET wrapper sends request with urllib.

Params:
url:(str) path to remote resource
timeout:(int) time (seconds) before giving up on unresposive hosts
randua:(bool) if randua=True urllib uses randomagent()
Throws:
  • HTTPError()
  • URLError()
  • UnhandledError()
Returns:

str(urllib.urlopen(url).read())
