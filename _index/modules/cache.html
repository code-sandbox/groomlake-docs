common.cache¶

lib.common.cache¶

exception lib.common.cache.CacheIOError¶

Raised when lib.common.cache.commit() cannot write to lib.common.control.CACHE_PATH

Inheritance:Exception()
exception lib.common.cache.TokenNotFoundError¶

Raised when lib.common.cache.get() finds zero / expired results

Inheritance:Exception()
lib.common.cache.clear()¶

Removes ALL files / folders from lib.common.control.CACHE_PATH

Returns:True when complete

Warning

This is destructive / irreversible, use with caution.

lib.common.cache.commit(f_token, data)¶

Save payload with filename token to lib.common.control.CACHE_PATH or raise Exception()

Params:
token:(str) md5 digest generated by file_token()
payload:(str) data to be written to disk
Throws:

CacheIOError()
Returns:

True when complete
lib.common.cache.file_token(url)¶

Used primarily for hashing URL’s and generating a unique filename, but it could also be used to check integrity of some arbitrary data, if that data is or can be cast to a string and has a known md5 value.

Params:
url:(str) some URL you want to hash
Returns:

str(md5hash(url))
lib.common.cache.get(f_token, decay=25)¶

Retrieve cached payload from lib.common.control.CACHE_PATH with filename matching token and mtime < decay or raise Exception()

Params:
token:(str) md5 digest generated by file_token()
decay:(int) time (minutes) before cached payloads are ignored
Throws:

TokenNotFoundError()
Returns:

str(payload)
