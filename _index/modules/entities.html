lib.core.entities¶

class lib.core.entities.ListItem(label='', meta=None, assets=None, query=None, _dir=True)¶

Base class containing common logic for parsing / serializing an API response entity (JSON, XML, etc.), this is an adopted model from xbmc/kodi that makes a reasonable base for an arbitrary media or subdirectory object.

Params:
label:(str) item name
meta:(dict) metadata dictionary
assets:(dict) asset dictionary
query:(dict) query dictionary
_dir:(bool) set to False for playable items

Note

ALL subclasses are responsible for populating member data.

querystring()¶

Generate a serialized string from current query. The querystring() contains information related to navigating around the different categories and subcategories of content on a host server.

The result defines expected behavior if / when this ListItem() is selected. (see example)

Returns:str
example¶
entity.query = {
    PluginArgument.ACTION:       PluginAction.BROWSE,
    PluginArgument.SECTION:      PluginSection.VOD,
    PluginArgument.CATEGORY_ID:  0
}

print(entity.querystring())
>>> "action=browse_category&xsection=vod&catgory_id=0"
