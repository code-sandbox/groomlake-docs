lib.core.entities¶

class lib.core.entities.ListItem¶

Base class containing common logic for parsing / serializing an API response entity (JSON, XML, etc.)

Note

subclasses are responsible for populating member data.

querystring()¶

Generate a serialized string from current querydict. The querystring() contains information related to navigating around the different categories and subcategories of content on a host server.

It should contain any data needed to make the request for the next payload that is NOT contained in the descriptive metadata (e.g. numeric id’s).

Returns:str
